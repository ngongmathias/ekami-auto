## üí° Pro Tip: Use URL Parameters

Unsplash allows image transformations via URL:
- `?w=800` - Set width to 800px
- `?flip=h` - Flip horizontally (creates mirror view)
- `?crop=entropy` - Smart crop
- `?auto=format` - Auto format

**Example:**
```
Original: https://images.unsplash.com/photo-123456
Flipped: https://images.unsplash.com/photo-123456?flip=h
Smaller: https://images.unsplash.com/photo-123456?w=600
```
